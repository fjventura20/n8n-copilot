## 💡 What It Does
n8n Copilot helps you:
- Create new workflows via plain text
- Edit existing workflows with natural language commands
- Integrate with various APIs and services
- Automate repetitive tasks
- MORE TO COME!!

## 🛠️ Features
- **Natural Language Processing:** Create and modify workflows using plain text commands.
- **API Integration:** Seamlessly connect to various APIs and services.
- **Automation:** Automate repetitive tasks and workflows.
- **Open Source:** Contribute, improve, and use the project for free.

## 📋 Installation
To get started with n8n Copilot, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/n8n-copilot.git
   cd n8n-copilot
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the application:**
   ```bash
   npm start
   ```

## 🚀 Usage
Once the application is running, you can start creating and editing workflows using natural language commands. Here are some examples:

- **Create a new workflow:**
  ```plaintext
  Create a workflow that sends an email when a new GitHub issue is created.
  ```

- **Edit an existing workflow:**
  ```plaintext
  Add a step to the workflow that logs the issue details.
  ```

## ⚙️ Configuration
To configure the AI providers and n8n integration, follow these steps:

1. **Open the Extension Settings:**
   Click on the n8n Copilot extension icon in your browser's toolbar.

2. **Open the Settings Panel:**
   Click on the gear icon (⚙️) to open the settings panel.

3. **Select AI Provider:**
   Choose between OpenAI and Anthropic as your AI provider.

4. **Enter API Keys:**
   - **OpenAI:** Enter your OpenAI API key in the designated field.
   - **Anthropic:** Enter your Anthropic API key in the designated field.

5. **Configure n8n Integration:**
   - **n8n API URL:** Enter the URL of your n8n instance (e.g., `https://your-n8n-instance.com`).
   - **n8n API Key:** Enter your n8n API key.

6. **Save Settings:**
   Click on the "Save Settings" button to save your configuration.

## 🔧 Load the Extension
To load the n8n Copilot extension into n8n, follow these steps:

1. **Open n8n:**
   Launch the n8n application.

2. **Access the Extensions Menu:**
   Click on your profile icon in the top-right corner and select "Extensions".

3. **Load the Extension:**
   Click on the "Load Extension" button and select the `n8n-copilot` directory from your cloned repository.

4. **Enable the Extension:**
   Once loaded, toggle the switch to enable the n8n Copilot extension.
